  <h1 align="center" style="border-bottom: none"><b>Exercise 6</b> - Moogle</h1>

  <p align="center">
<b>Moogle; Search & Rank Web Pages</b> is the sixth assignment I completed in the <b>Introduction to Computer Science</b> course at HUJI.
<br>
In this exercise, you’ll build a mini search engine that crawls a set of web pages, computes link-based rankings using PageRank, and enables keyword search using inverted index structures. Along the way, you’ll work with HTTP requests, HTML parsing (via BeautifulSoup), dictionaries, and command-line tools.
